# Contributing

Terima kasih sudah membantu!

## Alur
- Buat branch: `feat/...` atau `fix/...`
- Commit kecil & jelas, sertakan deskripsi perubahan.
- Jika menyentuh alur utama, jelaskan langkah uji (test) atau screenshot sheet.

## Panduan Kode
- Pertahankan nama fungsi publik (dipakai menu/trigger).
- Tambahkan komentar singkat pada fungsi baru.
- Jangan memasukkan API key / kredensial pada kode atau README.

## Rilis
- Pastikan `setupAll()` dan `runAllNow()` berjalan di spreadsheet baru.
- Update README jika ada variabel Script Properties baru/diubah.
